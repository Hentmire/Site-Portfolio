Star Wars Application

StarWarsApp

Также по имени персонажа может осуществляеться поиск.

Currencies Rates Application

CurrencyRateApp

Также можно указать период в 2018 года, за который необходимо отобразить данные.

График может быть построен как для одной из указанных валют, так и для нескольких.

Для курса доллара, евро и российского рубля можно построить график изменения курса за 2018 год.

Приложение Currencies Rates использует API Нац.банка РБ. Оно отображает текущие курсы валют.
